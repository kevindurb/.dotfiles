These are my **new** rc/config files for my development system.

# Requirements
- `stow`
- `nvim`
- `tmux`
- `zsh`
- `antigen` (`curl -L git.io/antigen > ~/antigen.zsh`)
- `alacritty` [Install Alacritty](https://github.com/alacritty/alacritty#installation)
- `starship` [Install Starship](https://starship.rs/guide/#%F0%9F%9A%80-installation)

# Setup
1) clone to ~/.dotfiles
2) `cd ~/.dotfiles`
3) `make`
